# Cursive Generators

A free online cursive text generator tool for students, 
social media users, and anyone who loves stylish fonts.

## 🔗 Live Site
👉 [cursivegenerators.us](https://cursivegenerators.us)

## ✨ Features
- Convert normal text to cursive fonts instantly
- One click copy
- Multiple cursive & fancy font styles
- No sign-up, no ads, no downloads
- Mobile friendly — works on all devices

## 🛠️ Built With
- HTML
- CSS  
- JavaScript

## 👥 Who Is It For?
- Students styling project titles & headings
- Social media users creating unique Instagram & WhatsApp bios
- Teachers making decorative worksheets
- Anyone who loves beautiful typography

## 📌 How To Use
1. Open [cursivegenerators.us](https://cursivegenerators.us)
2. Type your text
3. Copy your cursive result
4. Paste anywhere!
